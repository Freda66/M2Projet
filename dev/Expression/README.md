Addition:

A + B -> B + A OK

A + (B + C) -> (A + B) + C, (A + C) + B OK

Multiplication

A * B -> B * C

A * (B * C) -> (A * B) * C, (A * C) * B

Division

A * ( B / C ) -> ( A * B ) / C , B * ( A / C )

( A * B ) / C -> A * ( B / C ), B * ( A / C )

A / B * C / D -> ( A * C )/ ( B * D )

A + (B / C) -> ( (A * C) + B ) / C


Distributivité 

( A + B ) * C -> ( C * A ) + ( C * B )

( A - B ) * C -> ( C * A ) - ( C * B )

( C * A ) + ( C * B ) -> ( A + B ) * C 

( C * A ) - ( C * B ) -> ( A - B ) * C
