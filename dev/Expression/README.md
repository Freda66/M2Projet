Liste des Transformations a faire :

Addition:

A + B -> B + A OK
A + (B + C) -> (A + B) + C, (A + C) + B OK

Multiplication

A * B -> B * C
A * (B * C) -> (A * B) * C, (A * C) * B 

Distributivité 
( A + B ) * C -> ( C * A ) + ( C * B )
( A - B ) * C -> ( C * A ) - ( C * B )

( C * A ) + ( C * B ) -> ( A + B ) * C 
( C * A ) - ( C * B ) -> ( A - B ) * C 


Identités remarquables

( A + B ) * ( A - B ) -> A² + B²
( A + B ) * ( A + B ) -> A² + ( ( 2 * A) * B) + B²
( A - B ) * ( A - B ) -> A² - ( ( 2 * A) * B) + B²
