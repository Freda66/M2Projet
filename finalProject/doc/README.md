Liste des Transformations :

Addition:

A + B -> B + A

A + ( B + C ) -> ( A + B ) + C, ( A + C ) + B

Multiplication

A * B -> B * A

A * (B * C) -> (A * B) * C , (A * C) * B

Division

A * ( B / C ) -> ( A * B ) / C , B * ( A / C )

( A * B ) / C -> A * ( B / C ) , B * ( A / C )

( A / B ) * ( C / D ) -> ( A * C ) / ( B * D )

A + ( B / C ) -> ( ( A * C ) + B ) / C

Distributivité

( A + B ) * C -> ( C * A ) + ( C * B )

( A - B ) * C -> ( C * A ) - ( C * B )

( C * A ) + ( C * B ) -> ( A + B ) * C

( C * A ) - ( C * B ) -> ( A - B ) * C

Identités remarquables

( A + B ) * ( A - B ) -> A² + B²

( A + B ) * ( A + B ) -> A² + ( ( 2 * A ) * B ) + B²

( A - B ) * ( A - B ) -> A² - ( ( 2 * A ) * B ) + B²
